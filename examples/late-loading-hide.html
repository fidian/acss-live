Late Loading - Hide

Load the script late (after elements are in the DOM) and avoid a flicker while CSS styles are generated by initially hiding the body. To accomplish this, you need to have one style loaded in the head (either embedded in the HTML or in another stylesheet) that has body { display: none; }, then add class="D(b)" to your body tag.
