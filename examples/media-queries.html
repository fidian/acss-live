Media Queries

There are several media queries built into ACSS-Live. When you resize your screen, various elements will be indicated so you can see what applies.

Because of how ACSS-Live works, order does not matter. There is no default value and arrangements like D(n) D(b)--p will not work. You will need to use a combination of selectors when you want to conditionally present or style content, such as D(n)--_p D(b)--p. Another way is to use a higher specificity, such as D(n) x_D(b)--p and that will look for any parent element with class="x", which is why this example has that class on the body element.

Currently, this is using PRINT SCREEN media. It's matching a SMALL MEDIUM LARGE DESKTOP width device. Your system prefers DARK LIGHT (COLOR_SCHEME_NOT_SUPPORTED) theme.

In the table below, the "Active" column lists the atRule that is active currently. As you change the size of the browser window, view this page in a print preview, or perform other manipulations, the displayed text in the Active column will change. The atRule is placed at the end of a ACSS class name. A leading underscore applies the reversed form. For example, D(n)--p will apply "display: none" to the element while printing, and D(n)--_p will apply "display: none" when the element is not printing.

Active Media Query Description
--dk --_dk neither --dk nor --_dk @media (prefers-color-scheme:dark) Applied when the user wants a dark theme.
--lt --_lt neither --lt nor --_lt @media (prefers-color-scheme:light) Applied when the user wants a light theme.
--p --_p @media print Only applied when printing or showing the print preview.
--s --_s @media (max-width: 480px) Small screens, like phones in portrait mode.
--sm --_sm @media (max-width: 768px) Small or medium screens, like phones in any direction and tablets in portrait mode.
--m --_m @media (min-width: 480.0001px) and (max-width: 768px) Phones in landscape mode, tablets in portrait mode.
--sml --_sml @media (max-width: 1024px) Most mobile devices.
--ml --_ml @media (min-width: 480.0001px) and (max-width: 1024px) Mobile devices except phones in portrait mode.
--l --_l @media (min-width: 768.0001px) and (max-width: 1024px) Tablets in landscape mode.
