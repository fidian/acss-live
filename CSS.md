Supported CSS
=============

This is a list of all supported CSS in the current version of the library.

Global Default Values
---------------------

These can be used for any CSS rule.

* `inh` → `inherit`
* `in` → `initial`
* `rev` → `revert`
* `rel` → `revert-layer`
* `un` → `unset`
